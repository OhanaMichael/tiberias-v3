⚙️ איך ה-Excel מעדכן את ה-Sheet?
כדי שהעלאת Excel תכתוב ישירות ל-Google Sheet, יש להגדיר Apps Script Webhook ב-Sheet שלך.

ב-Google Sheets: הרחבות ← Apps Script
הדבק את הקוד מהקובץ apps-script.gs בפרויקט → שמור → פרוס כ-Web App (Anyone)
העתק את ה-URL שתקבל אל APPS_SCRIPT_WEBHOOK_URL ב-Netlify Environment Variables.

ללא Webhook — ה-Sheet יישאר ללא שינוי. כל שינוי ישיר ב-Sheet עצמו עדיין יפעל כרגיל.
